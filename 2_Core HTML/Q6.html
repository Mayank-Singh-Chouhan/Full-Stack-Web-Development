1. HTML

    Primary Use Cases Key Features/Benefits
    Building the structure of web pages
    • Allows for semantic markup
    • Simple and easy to learn
    • Compatible with all web browsers

  2. CSS

    Primary Use Cases Key Features/Benefits
    Styling web pages and elements
    • Provides extensive styling options
    • Separation of content and presentation
    • Responsive design with media queries

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    Adding interactivity to web pages
    Handling user interactions and form validation
    • Supports both front-end and back-end development
    • Large ecosystem of libraries and frameworks (e.g., React)
    • Can be used on both client and server-side (Node.js)
