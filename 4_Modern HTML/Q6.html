HTML vs HTML5

HTML HTML5
HTML4 had limited support for multimedia and
required third-party plugins.		 HTML5 includes native support for audio and video
elements, eliminating the need for plugins.
HTML4 didn't have semantic elements like <header>,
<nav>, <section>, etc.		 HTML5 introduced semantic elements that provide better
structure and meaning to web content.
HTML4 used presentational elements like <font>
and <center> for styling.		 HTML5 encourages the use of CSS for styling and
separates structure from presentation.
